Идея приложения в том, что в Украине 2100 года успешное экономическое развитите. Каждый в стране стал бизнесменом, и все нужно именно мое приложение для упрощения повседневной жизни.
Требуемые библиотеки:

pyqt5
pyperclip
plyer
CurrencyConverter
sqlite3
random
string
os
sys
inspect
datetime

запуск через app.py
